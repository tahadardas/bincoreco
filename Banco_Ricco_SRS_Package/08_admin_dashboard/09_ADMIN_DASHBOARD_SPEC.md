# Admin Dashboard Specification

## الهدف
لوحة تحكم واحدة لإدارة الموقع والتطبيق والعمليات اليومية.

## المستخدم الأساسي
المايسترو.

## الأقسام

### Dashboard
- طلبات اليوم.
- مبيعات اليوم.
- الأصناف الأكثر مبيعاً.
- طلبات قيد التحضير.
- تنبيهات المنتجات الناقصة.

### Orders
- Kanban أو Tabs حسب الحالة.
- فلترة بالتاريخ.
- بحث بالعميل أو رقم الطلب.
- تغيير الحالة.
- إلغاء مع سبب.

### Products
- CRUD.
- صور.
- ترجمات.
- أسعار.
- خيارات.
- Best seller.
- Maestro pick.

### Coffee Beans
- Roast profile.
- خيارات طحن.
- أوزان.
- حب/مطحون.

### B.R Coins
- رصيد العملاء.
- سجل الحركات.
- تعديل يدوي مع سبب.

### Reports
- الأصناف الأكثر مبيعاً.
- الطلبات اليومية.
- المبيعات.
- العملاء المتكررون لاحقاً.

## قواعد UX
- بحث وتصفية في كل صفحة.
- أزرار واضحة.
- لا تعديلات خطرة دون تأكيد.
- Audit log للتعديلات المهمة.
