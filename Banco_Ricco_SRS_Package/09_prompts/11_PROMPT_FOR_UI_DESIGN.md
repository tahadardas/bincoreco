# Prompt for UI/UX Design

Design Banco Ricco as a premium coffee bank experience, not as a generic restaurant app.

Use brand PDFs from:

```txt
assets/brand/
```

Visual direction:
- Black, gold, cream, coffee brown.
- Damascus heritage.
- Classic + modern.
- Premium.
- Elegant.
- Minimal ornamentation.

Core screens:
1. Splash.
2. Home.
3. Product Listing.
4. Product Detail.
5. Coffee Bean Detail.
6. Grind Selection.
7. Cart.
8. Pickup Tracking.
9. Banco Card QR.
10. B.R Coins.
11. Maestro Coffee Finder.
12. Admin Dashboard.
13. Orders Board.
14. Product Editor.
15. Reports.

UX priority:
- Avoid customer distraction.
- Highlight best sellers.
- Make pickup order extremely fast.
- Use Arabic RTL perfectly.
- Support English LTR.
