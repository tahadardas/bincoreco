# F06 — B.R Coins Loyalty

## Phase
Phase 2

## الهدف
بناء نظام ولاء باسم B.R Coins للزبائن اليوميين.

## قواعد الهوية
- كل الزبائن VIP.
- لا تعرض Bronze/Silver/Gold في الواجهة.
- يمكن استخدام تصنيف داخلي فقط للتقارير.

## النطاق
- رصيد نقاط.
- كسب نقاط بعد اكتمال الطلب.
- سجل حركات.
- تعديل يدوي من الإدارة مع سبب.
- قواعد نقاط قابلة للتعديل.

## قواعد العمل
- لا تضاف النقاط إلا بعد PICKED_UP أو COMPLETED.
- لا تعديلات يدوية بدون Audit Log.
- لا يسمح برصيد سالب إلا بصلاحية خاصة.

## Acceptance Criteria
- كل عميل يملك Loyalty Account.
- كل طلب مكتمل يولد نقاط حسب القاعدة.
- العميل يرى رصيده.
- المايسترو يرى السجل ويستطيع التعديل بصلاحية.
