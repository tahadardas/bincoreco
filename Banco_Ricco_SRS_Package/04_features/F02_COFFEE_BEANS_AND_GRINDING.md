# F02 — Coffee Beans & Grinding Options

## Phase
Phase 1

## الهدف
دعم بيع البن حب ومطحون حسب طريقة الاستخلاص.

## النطاق
- حب كامل.
- بن مطحون.
- درجات طحن حسب الاستخلاص.
- ربط خيارات الطحن بمنتجات البن.
- ظهور الخيارات في الطلب ولوحة التحكم.

## خيارات الطحن
```txt
WHOLE_BEAN       حب كامل
ESPRESSO         إسبريسو
TURKISH_SHAMI    شامي/تركي
V60              V60
AMERICAN_COFFEE  أمريكية
MOKA_POT         موكا بوت
FRENCH_PRESS     فرنش برس
COLD_BREW        كولد برو
```

## تدفق العميل
```txt
اختيار نوع البن
اختيار الوزن
اختيار حب كامل أو مطحون
إذا مطحون: اختيار طريقة الاستخلاص
إضافة للسلة
```

## قواعد العمل
- لا يسمح بإضافة بن مطحون بدون طريقة طحن.
- خيار الطحن ليس ملاحظة نصية؛ يجب أن يكون قيمة منظمة.
- إذا المنتج حب فقط لا تظهر خيارات الطحن.
- إذا المنتج يدعم الحب والمطحون يجب إظهار الاختيار بوضوح.

## شكل الطلب في لوحة التحكم
```txt
Product: V60 Colombia
Weight: 250g
Form: Ground
Grind: V60
Qty: 1
```

## Acceptance Criteria
- يستطيع العميل اختيار حب أو مطحون.
- الطلب يصل للموظف بتفاصيل الطحن.
- يمكن تعديل خيارات الطحن من لوحة التحكم.
