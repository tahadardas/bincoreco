# F03 — Roast Profile

## Phase
Phase 1 Basic / Phase 2 Enhanced

## الهدف
إظهار احترافية Banco Ricco في التحميص والنكهات دون إطالة.

## الحقول
```txt
roast_level
acidity_level
body_level
sweetness_level
bitterness_level
caffeine_level
flavor_notes

```

## قواعد المحتوى
- لا تكشف الخلطات السرية.
- اعرض Micro Profile واضح.
- لا تعرض حقول فارغة.

## أمثلة تطبيقية
- الإسبريسو والقهوة الأمريكية: توليفة بين الغامق والوسط الداكن.
- القهوة الشامية: خيارات متعددة مع الهال والمستكة.
- V60: كولومبي، إثيوبي، برازيلي.

## Acceptance Criteria
- يستطيع المايسترو إنشاء Roast Profile.
- تظهر معلومات التحميص في صفحة المنتج.
- تستخدم بيانات التحميص لاحقاً في اختيار المايسترو.
