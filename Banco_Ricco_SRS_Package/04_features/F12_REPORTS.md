# F12 — Reports

## Phase
Phase 1 Basic / Phase 2+

## أهم تقرير
الأصناف الأكثر مبيعاً.

## تقارير Phase 1
- مبيعات اليوم.
- عدد الطلبات.
- الطلبات حسب الحالة.
- أكثر المنتجات مبيعاً.
- الطلبات الملغاة.
- متوسط وقت التجهيز.

## تقارير Phase 2
- B.R Coins الممنوحة والمستخدمة.
- العملاء الأكثر تكراراً.
- نتائج اختيار المايسترو.
- QR scans.

## فلاتر
```txt
from_date
to_date
product
category
status
customer
currency
```

## Acceptance Criteria
- يستطيع المايسترو معرفة الأصناف الأكثر مبيعاً.
- التقارير تدعم فلترة تاريخية.
- الأرقام تعتمد على snapshots وليس الأسعار الحالية.
