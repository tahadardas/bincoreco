# F09 — Multi-language

## Phase
Phase 1

## اللغات
```txt
ar — Arabic RTL
en — English LTR
```

## النطاق
- ترجمة المنتجات.
- ترجمة التصنيفات.
- ترجمة البانرات.
- ترجمة محتوى الموقع.
- ترجمة إشعارات الطلب.
- دعم اتجاه الواجهة حسب اللغة.

## قواعد العمل
- العربية هي اللغة الافتراضية.
- الإنكليزية مهمة للسياح.
- لا تضع النصوص التجارية داخل الكود.
- لا تنشر منتج بدون لغة افتراضية مكتملة.

## Acceptance Criteria
- العميل يستطيع تبديل اللغة.
- الواجهة لا تنكسر في RTL/LTR.
- المنتجات والتصنيفات مترجمة من قاعدة البيانات.
