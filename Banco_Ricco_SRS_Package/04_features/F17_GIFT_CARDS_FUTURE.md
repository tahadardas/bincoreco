# F17 — Gift Cards

## Phase
Phase 4

## الهدف
بطاقات هدايا رقمية يمكن استخدامها في التطبيق أو المحل.

## الأنواع
```txt
AMOUNT
FREE_DRINK
SUBSCRIPTION_CREDIT
EXPERIENCE
```

## قواعد العمل
- كل بطاقة لها كود فريد أو QR.
- لها صلاحية.
- يسجل استخدامها.
- لا تنفذ قبل استقرار الدفع والطلبات.

## Acceptance Criteria
- يمكن إنشاء بطاقة.
- يمكن استردادها.
- الرصيد يتحدث بدقة.
