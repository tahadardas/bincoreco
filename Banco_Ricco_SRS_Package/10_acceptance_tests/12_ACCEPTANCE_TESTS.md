# Acceptance Tests

## Phase 1 Tests

### Product Catalog
- Admin can create product with Arabic/English names.
- Admin can set price.
- Customer sees active products only.
- Best sellers appear on Home.

### Coffee Beans
- Customer can choose whole bean.
- Customer can choose ground coffee.
- If ground coffee is selected, grind method is required.
- Order displays grind method to staff.

### Pickup Orders
- Customer can create pickup order.
- Default pickup time is 15 minutes.
- Staff can change status.
- Customer can see order status.
- Online payment is not shown.
- Delivery is not shown.

### Admin
- Maestro can manage products.
- Maestro can manage orders.
- Maestro can view top products report.

### Localization
- Arabic RTL works.
- English LTR works.
- Product translation displays correctly.

## Phase 2 Tests

### B.R Coins
- Completed order grants coins.
- Customer sees balance.
- Admin sees ledger.

### QR Card
- Customer has QR.
- Staff can scan QR.
- QR does not expose sensitive data.

### Maestro Finder
- Customer answers quiz.
- Recommendation appears with reason.
- Admin can edit questions.
