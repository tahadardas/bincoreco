# User Journeys

## 1. Pickup Order Journey

```txt
فتح التطبيق
اختيار اللغة
عرض الأكثر طلباً
اختيار منتج
تخصيص الخيارات
إضافة للسلة
اختيار وقت الاستلام
تأكيد الطلب
متابعة الحالة
استلام الطلب
كسب B.R Coins لاحقاً
```

## 2. Coffee Bean Journey

```txt
فتح قسم البن
اختيار نوع البن
قراءة Roast Profile
اختيار الوزن
اختيار حب كامل أو مطحون
إذا مطحون: اختيار طريقة الطحن
إضافة للسلة
تأكيد طلب الاستلام
```

## 3. Maestro Finder Journey

```txt
فتح اختيار المايسترو
الإجابة على 5-7 أسئلة
عرض التوصية
عرض سبب التوصية
إضافة المنتج للسلة
```

## 4. QR Card Journey

```txt
العميل يفتح Banco Card
يعرض QR
الموظف يمسح QR
تظهر النقاط والطلبات المفضلة
يتم تطبيق عرض أو احتساب نقاط
```

## 5. Admin Orders Journey

```txt
طلب جديد يظهر
الموظف يقبل الطلب
يبدأ التحضير
يحدد جاهز للاستلام
العميل يستلم
الطلب يغلق
```
